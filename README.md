# HealthConnect NGO - Healthcare Volunteer Registration Portal

## Project Overview
HealthConnect NGO is a simple healthcare volunteer registration web application designed to connect volunteers with healthcare initiatives and community outreach programs.

## Features
- Volunteer Registration Form
- AI-based Volunteer Role Recommendation
- AI Assistant Chatbot for FAQs
- Responsive User Interface
- NGO Information and Volunteer Opportunities

## Tech Stack
- HTML
- CSS
- JavaScript

## AI / Automation Idea
The application includes an AI-inspired volunteer recommendation system. Based on the volunteer's skills, the system automatically suggests a suitable volunteer role.

Examples:
- First Aid → Medical Assistance Volunteer
- Teaching → Health Awareness Educator
- Communication → Community Outreach Volunteer

## NGO Use Case
This platform helps NGOs efficiently register volunteers and assign them to suitable roles based on their skills and availability.

## How to Run
1. Download the project files.
2. Open index.html in a browser.
3. Fill the registration form and submit.
